# sui-docker
- Dockerfile for sui full node.
- [jionederfull/sui-docker](https://hub.docker.com/repository/docker/jionederfull/sui-docker/general)
```
sui-docker
├── README.md
├── dockerbuild
│   ├── Dockerfile
│   ├── README.md
│   └── config
│       ├── fullnode.yaml
│       └── genesis.blob
└── localbuild
    ├── Dockerfile
    ├── README.md
    ├── bin
    ├── build-sui-node.sh
    ├── config
    │   ├── fullnode.yaml
    │   └── genesis.blob
    └── sui(https://github.com/MystenLabs/sui)
```
